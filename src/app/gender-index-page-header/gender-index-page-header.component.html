מדד המגדר

כאן ניתן לראות את תוצאות מדד המגדר, הבוחן את אי השוויון בין נשים לגברים בישראל במגוון תחומי חיים ועל פני עשרות אינדיקטורים.
מדד המגדר בוחן את רמת האי שוויון המגדרי לאורך השנים, מציג מגמות שינוי ומאפשר לזהות באיזה תחום הפער המגדרי הוא העמוק ביותר.
